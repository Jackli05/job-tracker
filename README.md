# Job Application Tracker

A single-page web app to track job applications, built with vanilla HTML, CSS, and JavaScript.

**Live Demo**: [jackli05.github.io/job-tracker](https://jackli05.github.io/job-tracker)

## Features

- **Add & manage applications** — track company, position, date applied, resume version, and job description
- **Status tracking** — Applied, Online Assessment, Phone Screen, Interview, Offer, Rejected
- **Search & filter** — search by company/position, filter by status
- **Pagination** — navigate large application lists
- **Application Calendar** — month/week view with scheduled interview events
- **Stats dashboard** — live count of total applications, interviews, and offers
- **Import / Export** — save and load data as JSON
- **Autocomplete** — company name suggestions based on past entries
- **Local storage** — data persists across sessions in the browser

## Tech Stack

Vanilla HTML · CSS · JavaScript — no frameworks, no dependencies

## Usage

Open the live demo directly in your browser, or clone and open locally:
```bash
git clone https://github.com/Jackli05/job-tracker.git
open index.html
```
